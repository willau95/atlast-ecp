① Install the SDK

Choose your language and install with one command:

Python
TypeScript
pip install atlast-ecp

Requires Python 3.10+

npm install atlast-ecp-ts

Requires Node.js 18+

② Initialize Your Identity

Run the init command to create your cryptographic identity:

atlast init

This creates ~/.atlast/ with:

  • 🔑 Ed25519 keypair — Your agent's cryptographic identity
  • 🆔 DID — Decentralized Identifier (did:ecp:...)
  • ⚙️ Config — Server endpoint and preferences

③ Register & Get API Key

Register your agent with the ATLAST server to get an API key:

atlast register

④ Create Your First Record

Record an evidence chain entry with just 2 lines:

Python
TypeScript
from atlast_ecp.core import record # That's it — content is hashed locally, never leaves your device record_id = record("What is 2+2?", "The answer is 4")
import { record } from 'atlast-ecp-ts'; const recordId = record('What is 2+2?', 'The answer is 4');

What happens under the hood:

  • 1 Content is SHA-256 hashed locally
  • 2 Record is signed with your Ed25519 key
  • 3 Chain hash links to previous record
  • 4 Saved to local ~/.ecp/records/
  • 5 Queued for batch upload to server

⑤ Verify Your Chain

Check your evidence chain integrity anytime:

atlast log # View recent records atlast stats # Chain stats + integrity check atlast inspect <id> # Detailed record view

Or verify on-chain:

  • 🔗 Records are batched and anchored to Base via EAS
  • ✅ Verify at verify.weba0.com
  • 📊 Track everything in the Dashboard
