# ATLAST Protocol — Roadmap

## ✅ Shipped
- Python SDK v0.29+ (PyPI: atlast-ecp, 849+ tests)
- Go SDK (minimal ECP records, zero dependencies)
- ECP Reference Server (FastAPI, Railway)
- EAS on-chain anchoring (Base Mainnet, 63 attestations)
- Framework support: Claude Code, LangChain, CrewAI, AutoGen, OpenAI Agents SDK, Google ADK
- Universal proxy (OpenAI, Anthropic, OpenRouter, Groq, xAI, Mistral, DeepSeek, Cohere, Gemini)
- Local model support (Ollama, LM Studio, vLLM, HuggingFace transformers serve)
- Trust Score v2 (5-dimension legal-grade architecture)
- Score Pull API (third-party platforms pull scores from ATLAST server)
- Local Dashboard with On-Chain visualization, Token Analytics, Semantic Search
- Ed25519 cryptographic identity + SHA-256 hash chains
- Subagent recording (full tool call details)
- Streaming token extraction (OpenAI, Anthropic, Ollama, Gemini)
- Platform Integration Guide (third-party onboarding docs)
- EU AI Act compliance foundation
- Discord community + new user notifications

- Automated agent health monitoring and alerting
